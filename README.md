# Portfolio Website

Hello everyone!  
I'm Abin Varghese. Welcome to my portfolio website project.

**Tech Stack used:**  
- ReactJS  
- Tailwind CSS  
- Firebase  
- Material UI  
- SweetAlert2  

**Website Link:**  
[Portfolio Website](https://abin-portfolio-website.netlify.app/)

---

# Tutorial: Running the Project  

Here's a simple guide to run this project.  

## Prerequisites  

Ensure that you have already installed:  
- **Node.js**  

---

## Steps to Run the Project  

1. **Clone this project:**  

   ```bash  
   git clone https://github.com/Abinvarghese22/abin-portfolio-website.git  
   ```  

2. **Install all dependencies:**  

   ```bash  
   npm install  
   ```  
   Or if you encounter any dependency issues:  

   ```bash  
   npm install --legacy-peer-deps  
   ```  

3. **Run the project:**  

   ```bash  
   npm run dev  
   ```  

4. **Open in browser:**  

   Access the application through the link displayed in your terminal.  

---

## Creating a Production Build  

To create a production-ready build:  

1. Run the build command:  

   ```bash  
   npm run build  
   ```  

2. The build files will be saved in the `dist` folder. You can deploy this folder to your hosting service (e.g., Netlify).

---

## Notes  

If you encounter issues while running the project, ensure that:  
- Node.js is correctly installed
- You're in the correct project directory
- All dependencies are installed without errors

---

## Firebase Configuration  

To configure Firebase for this project:  

1. **Create a Firebase Project:**  
   - Go to the [Firebase Console](https://console.firebase.google.com/)
   - Create a new project or use an existing one

2. **Enable Firestore Database:**  
   - Create a database in Firestore
   - Set up the necessary collections for the portfolio

3. **Configure Firebase:**  
   - Go to Project Settings
   - Copy your Firebase configuration
   - Update the configuration in `src/firebase.js` and `src/firebase-comment.js`

4. **Update Security Rules:**  
   - Configure your Firestore security rules appropriately
